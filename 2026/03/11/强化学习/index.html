基本概念

  • state
  • action
  • policy
  • state transiction
  • reward
  • trajectory：轨迹是一条状态动作奖励链
  • return
    • discounted return
      • discount rate： γ ∈ (0, 1)
      • 使return变为有限值
      • 平衡远期和近期的收益（0近期收益，1远期收益）
  • MDP(Markov decision process)：马尔可夫决策过程

State Value and Bellman Equation

State Value

    • 含义：：在策略 π 下状态 s 的价值
    • ：第 t 步获得的奖励
    • γ：折扣因子（0~1）
    • E：期望（平均）
    • ：起始状态是
  • 从 s 开始，未来所有奖励的折扣和的期望值

Bellman Equationn

  • 描述了不同状态的state-value函数之间的关系
  • 由两部分组成：即时奖励和未来奖励
  • 每个state都有一个这样的公式

Action Value

  • 表示在某个状态下采取某个动作，然后继续按照策略 𝜋行动时能够获得的期望回报。

Optimal Policy and Bellman Optimality Equation

Optimal Policy（最优策略）

  • 对于一个策略来说，如果过对于所有状态s,该策略的回报期望（state value）均大于或等于其他任何策略，则称该策略为最优策略

BOE

  • 尝试所有动作a,选择期望回报最大的动作

  • 最优状态价值唯一，最优策略和最优轨迹不一定唯一
